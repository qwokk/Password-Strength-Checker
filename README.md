# Interactive Password Strength Checker
#### Веб-приложение для мгновенной проверки сложности пароля с динамическим визуальным откликом. Проект демонстрирует навыки чистой верстки, работы с регулярными выражениями и манипуляции DOM-деревом на чистом JavaScript.

# Стек технологий
### HTML5: 
#### Семантическая структура документа.
### CSS3:
#### Flexbox: Идеальное центрирование интерфейса.
#### Анимации: Плавные переходы (transition) для индикатора сложности.
#### Дизайн: Использование градиентов, фильтров (backdrop-filter) и кастомных шрифтов.
### JavaScript (Vanilla JS):
#### Обработка событий (input, click).
#### Регулярные выражения (RegExp) для валидации данных.
#### Логика управления состоянием UI через конструкцию switch.

# Основной функционал
#### Динамическая оценка: Анализ сложности пароля в реальном времени при вводе.
#### 5 критериев проверки: Наличие строчных и заглавных букв, цифр, специальных символов и минимальная длина (8 символов).
#### Визуальный индикатор (Strength Meter): Плавное заполнение и изменение цвета полоски прогресса в зависимости от уровня безопасности.
#### Show/Hide Password: Возможность переключения видимости вводимых символов.
#### Glassmorphism Design: Современный интерфейс с эффектом матового стекла, реализованный средствами CSS.

# Структура проекта
#### index.html — разметка приложения.
#### style.css — стили и визуальные эффекты.
#### script.js — логика проверки и интерактивность.
