# Interactive Password Strength Checker
#### Веб-приложение для мгновенной проверки сложности пароля с динамическим визуальным откликом. Проект демонстрирует навыки чистой верстки, работы с регулярными выражениями и манипуляции DOM-деревом на чистом JavaScript.

# Стек технологий
### HTML5: 
#### Семантическая структура документа.
### CSS3:
#### Flexbox: Идеальное центрирование интерфейса.
#### Анимации: Плавные переходы (transition) для индикатора сложности.
#### Дизайн: Использование градиентов, фильтров (backdrop-filter) и кастомных шрифтов.
### JavaScript (Vanilla JS):
#### Обработка событий (input, click).
#### Регулярные выражения (RegExp) для валидации данных.
#### Логика управления состоянием UI через конструкцию switch.
